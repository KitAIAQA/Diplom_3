## Дипломный проект. Задание 3: Автотесты для UI

### Автотесты для проверки веб-приложения Stellar Burgers.

### Установка зависимостей

> `pip install -r requirements.txt`

### Реализованные сценарии

#### Проверка основного функционала (`test_main_page.py`)
Проверка перехода по клику на «Конструктор»,
Проверка перехода по клику на «Лента заказов»,
Проверка если кликнуть на ингредиент, появится всплывающее окно с деталями,
Всплывающее окно закрывается кликом по крестику,
При добавлении ингредиента в заказ, увеличивается каунтер данного ингредиента
залогиненный пользователь может оформить заказ.

#### Раздел «Лента заказов» (`test_order_feed_page.py`)
Проверка если кликнуть на заказ, откроется всплывающее окно с деталями,
Проверка заказы пользователя из раздела «История заказов» отображаются на странице «Лента заказов»,
Проверка при создании нового заказа счётчик Выполнено за всё время увеличивается,
Проверка при создании нового заказа счётчик Выполнено за сегодня увеличивается,
Проверка после оформления заказа его номер появляется в разделе В работе.


### Запуск автотестов

Для Chrome (по умолчанию):
pytest

Для Firefox:
pytest --browser=firefox
